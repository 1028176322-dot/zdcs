#### 场景 2：Dump UI 树

```python
# 文件：core_engine/poco_connector/poco_connector.py (第 55-70 行)

def dump_ui_tree(self) -> Optional[Dict]:
    """
    获取当前界面的UI树
    :return: UI树 (dict)，失败返回None
    """
    if not self.poco:
        print("✗ Poco未连接，请先调用connect()")
        return None
    
    try:
        ui_tree = self.poco.dump()
        print(f"✓ 成功dump UI树，共{len(str(ui_tree))}字节")
        return ui_tree
    except Exception as e:
        print(f"✗ dump UI树失败: {e}")
        return None
```

**说明**：
- `poco.dump()` 获取当前界面的完整 UI 树（XML/JSON 格式）
- UI 树包含所有 UI 元素的信息：名称、位置、大小、文本等
- 用于后续的元素查找和 UI 验证

#### 场景 3：查找 UI 元素

```python
# 文件：core_engine/poco_connector/poco_connector.py (第 72-95 行)

def find_element(self, name: Optional[str] = None, text: Optional[str] = None, **kwargs):
    """
    查找UI元素
    :param name: 元素名称 (如 'ClickContent')
    :param text: 元素文本 (如 '探险家试炼')
    :param kwargs: 其他查询条件
    :return: Poco对象，未找到返回None
    """
    if not self.poco:
        print("✗ Poco未连接，请先调用connect()")
        return None
    
    try:
        if name:
            return self.poco(name)
        elif text:
            # 使用text属性查找
            return self.poco(text=text)
        else:
            print("✗ 必须提供name或text参数")
            return None
    except Exception as e:
        print(f"✗ 查找元素失败: {e}")
        return None
```

**使用示例**：
```python
# 按名称查找
element = poco('Button_Start')

# 按文本查找
element = poco(text='开始游戏')

# 按属性查找
element = poco(type='Button', enabled=True)
```

#### 场景 4：点击 UI 元素

```python
# 文件：core_engine/test_executor.py (第 84-101 行)

def _execute_click(self, action):
    """执行点击操作"""
    result = {'success': False, 'message': '', 'protocols': [], 'screenshot': None}
    
    # 解析点击目标
    target = self._parse_click_target(action)
    
    if not target:
        result['message'] = f"无法解析点击目标: {action}"
        return result
    
    try:
        # 查找并点击元素
        if self.poco(target).exists():
            self.poco(target).click()
            result['success'] = True
            result['message'] = f"成功点击: {target}"
        else:
            result['message'] = f"元素不存在: {target}"
            
    except PocoTargetTimeout as e:
        result['message'] = f"元素超时: {e}"
    except Exception as e:
        result['message'] = f"执行出错: {e}"
    
    return result
```

**说明**：
- 使用 `element.click()` 方法点击元素
- 支持等待元素出现：`poco(target).wait_for_appearance(timeout=10)`
- 支持超时处理：`PocoTargetTimeout` 异常

## pywin32 使用详解

**安装状态**：✓ 已安装 (版本 311)  
**用途**：Windows API 访问，用于查找游戏窗口

**安装包**：`pywin32` (311)

**在项目中的使用场景**：

### 场景 1：查找 Unity 窗口

```python
# 文件：visualize_clickable_elements.py (第 34-72 行)

import win32gui
import win32con

def find_unity_window() -> Optional[Tuple[int, int, int, int]]:
    """
    使用 Windows API 查找 Unity 窗口
    :return: (left, top, right, bottom) 或 None
    """
    try:
        # 存储找到的窗口句柄
        found_hwnd = [None]
        
        # 回调函数：枚举所有窗口
        def enum_callback(hwnd, lParam):
            # 获取窗口标题
            title = win32gui.GetWindowText(hwnd)
            
            # 检查是否是 Unity 窗口（标题包含 "Unity"）
            if 'unity' in title.lower():
                # 检查窗口是否可见
                if win32gui.IsWindowVisible(hwnd):
                    found_hwnd[0] = hwnd
                    return False  # 停止枚举
            
            return True  # 继续枚举
        
        # 枚举所有顶层窗口
        win32gui.EnumWindows(enum_callback, 0)
        
        if found_hwnd[0]:
            # 获取窗口矩形（屏幕坐标）
            left, top, right, bottom = win32gui.GetWindowRect(found_hwnd[0])
            
            print(f"✅ 找到 Unity 窗口: ({left}, {top}, {right}, {bottom})")
            return (left, top, right, bottom)
        
        print("⚠️ 未找到 Unity 窗口")
        return None
        
    except Exception as e:
        print(f"⚠️ Windows API 查找失败: {e}")
        return None
```

**功能说明**：
- 使用 Windows API 查找 Unity 游戏窗口
- 获取窗口位置和大小
- 用于智能裁剪截图到游戏区域

### 场景 2：窗口操作（可选）

## 常见问题

### Q1: POCO 连接失败怎么办？

**A**: 检查以下几点：
1. Unity 项目中是否已导入 POCO SDK
2. 游戏中是否挂载了 `PocoManager` 脚本
3. 游戏是否正在运行
4. 防火墙是否阻止了端口（默认 5001）

### Q2: Airtest 找不到设备怎么办？

**A**: 尝试以下方法：
1. 确认设备已连接（Windows/Android/iOS）
2. 使用正确的设备 URI
3. 检查设备驱动是否安装（Android）
4. 检查 USB 调试是否开启（Android）

### Q3: Pillow 图像处理失败怎么办？

**A**: 检查以下几点：
1. 图片文件路径是否正确
2. 图片格式是否支持（PNG、JPG 等）
3. 是否有文件读取权限
